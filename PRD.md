# PRD — ITCEN 실시간 신청곡 플랫폼

## 1. 프로젝트 개요

- **해결하려는 문제**: 워드 파일 기반 신청곡 수집 방식은 실시간 취합이 안 되고, 익명성이 없어 임직원들이 신청을 꺼리는 상황
- **배경**: 강의 중 신청곡을 놓치지 않고 바로바로 확인하고 싶음
- **기대 변화**: 링크 하나로 익명 접근 → 신청 즉시 강사에게 실시간 알림

## 2. 사용자 분석

- **주 사용자**: ITCEN 강의 참석 임직원 (~25명) / 수신자: 강사(본인)
- **사용 맥락**: 강의 중, PC 또는 모바일 브라우저로 링크 접속
- **현재 해결 방식과 한계**: 워드 파일로 수집 → 실시간 확인 불가, 익명성 없어 신청 꺼림

## 3. 기능 요구사항

- **핵심 기능**
  1. 곡 신청 폼 — 제목 + 가수명 입력 후 익명 제출
  2. 강사 실시간 알림 — 신청 즉시 강사 화면에 표시
  3. 링크 접속 — 로그인 없이 URL 하나로 바로 접근

- **차별점**: 로그인·개인정보 없이 링크만으로 익명 신청 가능

## 4. 비기능 요구사항

- **사용 환경**: 웹 (PC + 모바일 브라우저), Railway 배포
- **데이터 민감도**: 곡명·가수명만 수집 (개인정보 없음, 익명)
- **제약**: 간단하게, 빠르게
